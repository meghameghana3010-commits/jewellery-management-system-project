# 💎 Shri Jewellery – E-Commerce Jewellery Management System

## 📌 Project Overview
This project is an **E-Commerce Jewellery Management System** developed as part of the **Bachelor of Computer Applications (BCA)** curriculum.  
The system provides an online platform for browsing, selecting, and purchasing jewellery items with a smooth and user-friendly experience.


## ✨ Key Features
- User-friendly jewellery browsing (Gold, Silver, Diamond)
- Secure OTP-based user authentication
- Smart shopping cart with real-time price calculation
- Order placement, confirmation, and order tracking
- Virtual Try-On feature for previewing jewellery
- Admin-friendly and customer-friendly workflow
